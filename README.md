## 💻 About This Project

Welcome to the **Login Page Collection** — a creative pack of **10 different login page templates**, built with  
**HTML, CSS, JavaScript, and Bootstrap 5**.  

Each form includes **modern 2D & 3D animations**, **hover effects**, and **responsive layouts**, ideal for:

✅ Institute Websites  
✅ Student Projects  
✅ Demo Exams & Admin Dashboards  
✅ Web Design Learning Practice  

---

## 🧩 Key Features

🌈 **Responsive Design** – Works on all devices  
🎨 **2D & 3D Animations** – Attractive and interactive  
⚡ **All-in-One File** – Easy to use and modify  
💾 **Clean Code** – Built with HTML, CSS, JS & Bootstrap  
💡 **Ideal for Students** – Practice & learn frontend design
